## 🎮 About the Game

용감한 카피바라가 정글을 뛰어넘는 모험을 떠납니다!

- 정글을 누비는 액션 러너
- 보상형 광고로 리워드 획득
- 귀여운 동물들과의 상호작용
- 다양한 장애물과 맵 디자인
